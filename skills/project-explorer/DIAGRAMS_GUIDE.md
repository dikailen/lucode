# 项目可视化图表使用指南

## 图表功能概述

项目探索者技能现在支持生成多种可视化图表，帮助你更直观地理解项目结构和技术栈。

## 支持的图表类型

### 1. 📁 目录树结构图
- **功能**: 直观展示项目的目录层次关系
- **用途**: 理解项目的整体目录组织
- **特点**: 使用树状图清晰展示层级结构

### 2. 🍕 技术栈饼图
- **功能**: 显示项目使用的技术栈分布
- **用途**: 了解项目的技术组成比例
- **特点**: 饼图展示各技术占比，一目了然

### 3. 🏗️ 架构关系图
- **功能**: 展示系统的分层架构和数据流向
- **用途**: 理解系统的整体架构设计
- **特点**: 分层展示，包含组件示例

### 4. 📊 文件类型分布图
- **功能**: 统计和显示项目中的文件类型分布
- **用途**: 了解项目的文件组成结构
- **特点**: 柱状图+饼图双重展示

## 使用流程

### 1. 请求图表生成
```
用户：分析完项目后，能否生成一些图表帮助理解？
技能：我可以为您生成多种可视化图表，包括：
- 目录树结构图
- 技术栈饼图  
- 架构关系图
- 文件类型分布图

是否需要生成这些图表？
```

### 2. 用户确认
```
用户：是的，请生成所有图表
技能：好的，我将开始生成图表。这可能需要几分钟时间...
```

### 3. 图表生成
技能会：
- 扫描项目数据
- 生成图表
- 保存到项目目录
- 展示图表预览

### 4. 结果展示
```
生成的图表:
- directory_tree: /path/to/project/diagrams/directory_tree.png
- tech_stack: /path/to/project/diagrams/tech_stack_pie.png
- architecture: /path/to/project/diagrams/architecture_diagram.png
- file_types: /path/to/project/diagrams/file_type_distribution.png
```

## 依赖要求

生成图表需要安装以下Python库：
```bash
pip install matplotlib networkx numpy
```

## 高级用法

### 自定义图表类型
```
用户：我只想看目录树图和架构图
技能：好的，我将只生成您需要的图表类型
```

### 特定深度分析
```
用户：我想深入查看src目录的结构

```

### 对比分析
```
用户：能否对比两个不同版本的目录结构？

```

## 注意事项

1. **性能考虑**: 生成图表可能需要几分钟时间，特别是大型项目
2. **内存使用**: 复杂图表可能需要较多内存
3. **文件权限**: 确保有写入图表文件的权限
4. **图表大小**: 生成的图片会保持高分辨率(300dpi)

## 故障排除

### 常见问题

**Q: 图表生成失败，提示缺少依赖**
A: 请安装必要的Python库：
```bash
pip install matplotlib networkx numpy
```

**Q: 生成的图表模糊不清**
A: 图表默认生成300dpi分辨率，如果还是模糊，可以调整DPI参数

**Q: 图表文件过大**
A: 可以通过调整图形大小和压缩来优化文件大小

**Q: 某些目录显示不完整**
A: 路径太长的目录会被截断，可以通过调整显示参数来优化

### 优化建议

1. 对于大型项目，建议分批生成图表
2. 可以指定只生成特定类型的图表
3. 调整图形尺寸以适应不同的显示需求
